## 1.3 基本操作入门

### 1.3.1 Cursor 基本操作

1. 文件操作
```plaintext
- 创建文件：Cmd/Ctrl + N
- 保存文件：Cmd/Ctrl + S
- 查找文件：Cmd/Ctrl + P
- 切换文件：Cmd/Ctrl + Tab
```

2. 编辑操作
```plaintext
- 代码补全：Tab
- 智能提示：Cmd/Ctrl + Space
- 快速修复：Alt + Enter
- 重构操作：Cmd/Ctrl + R
```

3. AI 辅助操作
```plaintext
- 触发 AI：Cmd/Ctrl + I
- 代码生成：Cmd/Ctrl + Enter
- 代码解释：Alt + /
- 问题诊断：Cmd/Ctrl + Shift + D
```

### 1.3.2 Windsurf 基本操作

1. 项目操作
```plaintext
- 创建项目：New Project
- 导入项目：Import
- 项目配置：Settings
- 团队管理：Team
```

2. 代码生成操作
```plaintext
- 选择模板：Templates
- 配置参数：Config
- 生成代码：Generate
- 预览修改：Preview
```

3. 协作操作
```plaintext
- 任务创建：New Task
- 代码评审：Review
- 团队讨论：Discussion
- 知识共享：Share
```
