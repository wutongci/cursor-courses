# 第一章 基础入门

## 1.1 开发环境准备

### 1.1.1 系统要求
- 操作系统：
  - Windows 10/11（64位）
  - macOS 10.15+（支持 Intel 和 Apple Silicon）
  - Linux（Ubuntu 20.04+, Debian 10+）
- 硬件要求：
  - CPU：推荐 4核及以上
  - 内存：最低 8GB，推荐 16GB 以上
  - 硬盘空间：至少 1GB 可用空间（建议 SSD）
  - 显示器：推荐 1920x1080 及以上分辨率
- 网络要求：
  - 稳定的互联网连接（最低 5Mbps）
  - 能够访问 OpenAI API 服务器
  - 推荐使用代理以提升连接稳定性

### 1.1.2 前置软件环境
1. 必需软件
   - Git（版本控制）
   - Node.js（v14+，推荐使用 LTS 版本）
   - Python（3.7+，用于某些扩展功能）

2. 推荐软件
   - nvm（Node.js 版本管理器）
   - pyenv（Python 版本管理器）
   - Docker（容器化环境）

### 1.1.3 详细安装步骤

1. Cursor 安装配置
   ```bash
   # Windows
   1. 从 https://cursor.sh 下载 Windows 安装包（.exe）
   2. 双击运行安装程序，按提示完成安装
   3. 首次运行需要管理员权限

   # macOS
   1. 访问 https://cursor.sh 下载 .dmg 文件
   2. 打开 .dmg 文件，将 Cursor.app 拖入 Applications 文件夹
   3. 在 Security & Privacy 设置中允许运行

   # Linux
   1. 下载 .AppImage 或 .deb 包
   2. 为 .AppImage 添加执行权限：chmod +x Cursor.AppImage
   3. 运行安装程序
   ```

2. API Key 配置
   - 注册 OpenAI 账号
   - 生成 API Key
   - 在 Cursor 设置中配置 API Key
   - 验证连接状态

## 1.4 实战案例

### 1.4.1 Web 应用开发
1. 项目初始化
   ```bash
   # 创建项目
   mkdir web-app
   cd web-app
   
   # 初始化 Node.js 项目
   npm init -y
   
   # 安装依赖
   npm install express typescript @types/express
   npm install -D ts-node nodemon
   ```

2. 配置 TypeScript
   ```json
   // tsconfig.json
   {
     "compilerOptions": {
       "target": "es6",
       "module": "commonjs",
       "outDir": "./dist",
       "strict": true,
       "esModuleInterop": true
     }
   }
   ```

3. 创建服务器
   ```typescript
   // src/index.ts
   import express from 'express';
   
   const app = express();
   const port = 3000;
   
   app.get('/', (req, res) => {
     res.send('Hello from Cursor!');
   });
   
   app.listen(port, () => {
     console.log(`Server running at http://localhost:${port}`);
   });
   ```
