# Cursor AI 移动端开发课程

## 课程概述
本课程面向移动端开发工程师，包括Android、iOS和跨平台开发的AI辅助开发实践。

## 目录结构
1. Android开发
   - Kotlin与Android开发
   - Jetpack Compose实践
   - 性能优化与调试
   - AI辅助开发工具

2. iOS开发
   - Swift编程实践
   - SwiftUI应用开发
   - iOS架构设计
   - AI开发助手应用

3. 跨平台开发
   - Flutter高级开发
   - React Native实战
   - 混合开发方案
   - 性能优化策略

4. 移动端架构
   - 架构设计模式
   - 数据存储方案
   - 网络框架选择
   - 组件化开发

5. 质量与发布
   - 自动化测试
   - CI/CD流程
   - 应用发布流程
   - 版本管理策略

## 技术栈
- Android: Kotlin, Jetpack
- iOS: Swift, SwiftUI
- 跨平台: Flutter, React Native
- 工具链: Git, CI/CD
- 测试框架
